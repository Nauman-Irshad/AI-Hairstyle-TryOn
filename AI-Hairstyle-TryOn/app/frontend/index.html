Technical details 

        

      

    


    

      

          
Pipeline

        
Fast path: rembg cutout. Third column (hair removed) uses BiSeNet + LaMa — load models with SKIP_HAIR_PIPELINE=0. Full hairstyle try-on uses the same bundle.

      

      

        
Backend

        
Start with python run.py from the project root. Install rembg for background removal. Open http://127.0.0.1:8000.

      

      

        
Diagnostics

        
Header shows live API health. Errors below list HTTP status, server message, and raw response when possible.
